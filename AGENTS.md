# AGENTS.md

面向自动化助手与贡献者的**入口页**：只保留两件事——**项目目标**与**文档地图**（其余说明均在 `docs/` 下按主题拆分）。

## 1. 项目目标

TodoList 单体仓库交付一套可运行的待办应用：**FastAPI** 后端（PostgreSQL、Alembic）与 **Android** 客户端（Jetpack Compose）；在多租户隔离下提供待办与认证等能力，并可选支持语音转文字（STT）流式通道。详细表述见 [docs/project/goals.md](docs/project/goals.md)。
